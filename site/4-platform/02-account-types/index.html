Типы аккаунтов

Тип аккаунта определяет, данные какого сервиса будут храниться в аккаунте.

MarketAut поддерживает следующие типы аккаунтов:

  1. Wildberries — API-токен Wildberries

Wildberries

Позволяет работать с данными магазина через API Wildberries.

Доступно

  • Карточки товаров;
  • Цены, скидки, акции;
  • Поставки и фулфилмент;
  • Вопросы и отзывы покупателей;
  • Аналитические и финансовые отчёты.

Получение токена

Перед созданием аккаунта необходимо получить API-токен Wildberries.
Для получения токена необходимо иметь доступ владельца к личному кабинету Wildberries.

  1. В личном кабинете Wildberries перейдите в раздел «Интеграции по API»;
  2. Нажмите Создать токен;
  3. Выберите Для интеграции вручную;
  4. Выберите Базовый токен;
  5. Укажите необходимые права доступа (например, Контент, Цены и скидки);
  6. Нажмите Создать токен;
  7. Скопируйте созданный токен и добавьте его в MarketAut.
