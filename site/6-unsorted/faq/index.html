Типичные проблемы

MCP не отображается со статусом Running

Причины

  • диаграмма не запущена
  • token указан неверно
  • diagram настроена с ошибками
  • отсутствует соединение блоков
  • Claude не был перезапущен

Решение

  1. Проверьте что diagram запущена
  2. Проверьте корректность token
  3. Убедитесь что все блоки сохранены
  4. Проверьте соединения:
    • call → call
    • tools → tools
  5. Полностью перезапустите Claude

Claude не использует tools

Причины

  • tool отключен через Blocked
  • diagram выключена
  • MCP не подключен
  • отсутствует статус Running
  • tool не соединен с MCPv2
  • отсутствует аккаунт у блока

Решение

  1. Проверьте статус Running
  2. Проверьте настройки Connectors
  3. Убедитесь что tool не находится в статусе Blocked
  4. Проверьте соединения diagram
  5. Убедитесь что выбран аккаунт у block

Claude отвечает без tools

Claude не всегда обязан использовать tools.

Если вопрос не требует получения данных через MCP, Claude может ответить самостоятельно.

Примеры

  • генерация текста
  • помощь с документацией
  • объяснение функций
  • общие вопросы

Решение

Формулируйте запрос более конкретно.

Например:
- получи список товаров
- покажи цены
- обнови карточку
- создай карточку товара

Красная точка у блока

Красная точка означает что block настроен не полностью.

Причины

  • не выбран аккаунт
  • не заполнены обязательные поля
  • настройки не сохранены

Решение

  1. Нажмите на block
  2. Заполните обязательные поля
  3. Выберите аккаунт
  4. Нажмите Сохранить

Красный крестик на соединении

Красный крестик означает ошибку соединения blocks.

Решение

  1. Обновите страницу
  2. Повторно соедините blocks
  3. Проверьте правильность соединения
  4. Перезапустите diagram

Неправильное соединение blocks

Правильные соединения

External Endpoint → MCPv2

Соединение:
- call → call

Wildberries tools → MCPv2

Соединение:
- tools → tools

Ошибка

Если соединения выполнены неверно:
- tools могут не отображаться
- MCP может не запускаться
- Claude не сможет использовать tools

Tool не отображается в Claude

Причины

  • diagram выключена
  • MCP не подключен
  • block не соединен с MCPv2
  • отсутствует статус Running

Решение

  1. Проверьте что diagram запущена
  2. Проверьте Running
  3. Проверьте соединения
  4. Перезапустите Claude

Token не работает

Причины

  • token указан неверно
  • token устарел
  • token вставлен с ошибкой
  • отсутствует authorization header

Решение

  1. Создайте новый token
  2. Откройте:
    • Settings
    • Developer
    • Edit Config
  3. Замените token
  4. Перезапустите Claude

Diagram выключена

После остановки diagram:
- MCP перестанет работать
- tools станут недоступны
- статус Running может исчезнуть

Решение

  1. Откройте diagram
  2. Нажмите Play
  3. Перезапустите Claude
