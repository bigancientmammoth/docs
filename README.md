# Правила

## Именование файлов

1. Только **строчные** символы. ЗАГЛАВНЫЕ использовать нельзя. Причина: различие в интерпретации между разными ОС (Windows / MacOS)
2. Только английский буквы, дефисы `-` и подчеркивания `_`

## Контент

Списки оформляются через цифру с точкой. 

Например `1.` 

Варианты вроде `1)` использовать НЕЛЬЗЯ. Причина: различные парсеры Markdown по-разному интерпретируют списки, а стандарт предписывает точку.
