# Регистрация

Доступные способы регистрации:

- Учетная запись Google

Для регистрации:

1) Перейдите на [Главную страницу](/)
2) Выберите **Продолжить с Google**
![](img/register.png)
3) После подтверждения в Google вы окажетесь на главной странице **Marketaut**


# Вход

Вход происходит автоматически при посещении сайта, если вы на нем зарегистрированы. 
Если это не произошло (например, вы заходите с нового устройства), то следуйте инструкциям из раздела **Регистрация** (шаги для входа полностью аналогичны).
