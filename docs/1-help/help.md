# Часто задаваемые вопросы.


## С какими нейросетями работает MarketAut?

MarketAut работает с ИИ-агентами, поддерживающими протокол [MCP](https://modelcontextprotocol.io/docs/getting-started/intro)

**MCP** — это протокол для взаимодействия ИИ-агентов с внешними системами. Его поддерживают многие крупные поставщики ИИ, включая:

1. Claude (Anthropic)
2. ChatGPT (OpenAI)
3. Gemini (Google)
4. Deepseek (Hangzhou DeepSeek Artificial Intelligence)

## Нужен ли мне аккаунт у поставщика ИИ?

Да, нужен. MarketAut **НЕ** предоставляет собственные ИИ-модели. Платформа позволяет подключить аккаунт выбранного **поставщика ИИ** (Claude, ChatGPT, Gemini и других) к магазину Wildberries через протокол MCP.

Вопросы регистрации, оплаты подписки и получения доступа к ИИ-сервисам пользователь решает самостоятельно.
