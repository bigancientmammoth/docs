# Часто задаваемые вопросы


## С какими ИИ-сервисами работает MarketAut?


**MarketAut** работает с ИИ-агентами, поддерживающими протокол [MCP](https://modelcontextprotocol.io/docs/getting-started/intro).

MCP — это протокол, который позволяет языковой модели подключаться к внешним инструментам и данным. Благодаря этому модель может не только отвечать на вопросы, но и выполнять действия через инструменты MarketAut.

На текущий момент подключение к MarketAut доступно для **ChatGPT** и **Claude**. Остальные языковые модели из списка рассматриваются как потенциальные варианты для дальнейшей проверки.

## Нужен ли мне аккаунт у поставщика ИИ?

Да, нужен. **MarketAut не предоставляет собственные языковые модели**. Платформа позволяет подключить аккаунт выбранного **поставщика ИИ** (Claude, ChatGPT, Gemini и других) к магазину **Wildberries** через протокол MCP.

Пользователь самостоятельно регистрируется у поставщика ИИ, оплачивает подписку и получает доступ к выбранному сервису.
