# Общая информация

В этом разделе описано подключение ChatGPT к платформе MarketAut.

Подключение выполняется через протокол MCP. После подключения ChatGPT сможет использовать инструменты и ресурсы, доступные в вашей диаграмме MarketAut.

# Подключение

1) В левом нижнем углу окна ChatGPT нажмите на значок профиля и выберите **Настройки**.

![](img/gpt/profile_open.png)

2) Перейдите в раздел **Приложения**.
3) Нажмите **Создать приложение**.

![](img/gpt/app_create.png)


Откроется окно создания приложения.

Заполните следующие значения:
**Название** — любое название на ваш выбор, например `MarketAut`
**Подключение** — URL-адрес MCP-сервера.

Скопируйте ссылку из блока **MCP** в диаграмме MarketAut.

Например: `https://marketaut.ru/mcp/myMCP`

**Аутентификация** — `OAuth`

![](img/gpt/connector_dialog.png)

После заполнения полей установите флажок подтверждения и нажмите **Создать**.

![](img/gpt/confirm_creation.png)

4) После создания приложения нажмите **Войти через  MarketAut**
5) Вы будете перенаправлены на страницу авторизации MarketAut
6) В открывшемся окне предоставления доступа нажмите **Разрешить доступ**.

![](img/gpt/oauth_consent.png)

После завершения проверки приложение будет подключено к ChatGPT.

В разделе **Приложения** можно просмотреть информацию о приложении и список доступных инструментов.

![](img/gpt/tools_available.png)
# Работа с ChatGPT

## Проверка подключения

В новом диалоге ChatGPT нажмите кнопку **+** рядом с полем ввода сообщения.

Выберите:

**Больше → Режим агента → MarketAut**

![](img/gpt/add_marketaut.png)

После выбора агента MarketAut будет подключён к текущему диалогу.

Для проверки отправьте запрос например:

> Привет! Выведи список моих карточек WB.

Если подключение настроено корректно, ChatGPT сможет обратиться к инструментам MarketAut и получить данные из вашего магазина Wildberries.

![](img/gpt/ask.png)

## Обновление инструментов

После изменения списка доступных инструментов в диаграмме MarketAut необходимо обновить приложение в ChatGPT.

Для этого откройте:

**Настройки → Приложения → MarketAut**

Нажмите кнопку **Обновить**.

![](img/gpt/tools_update.png)

После обновления новые инструменты станут доступны в ChatGPT.
