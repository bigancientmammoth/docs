

# Общая информация

В этом разделе описано подключение ИИ-агентов (Claude, ChatGPT, Gemini и др.) к платформе MarketAut.

В качестве примера используется **Claude** от Anthropic. При этом большинство современных ИИ-агентов поддерживают протокол [MCP](https://modelcontextprotocol.io/docs/getting-started/intro) и могут подключаться через платформу MarketАut к вашему магазину Wildberries.


# Подключение

1) Запустите приложение **Claude for Desktop**
2) В разделе Chats выберите **Customize**

![](img/claude/customize.png)

3) В открывшемся окне выберите **Connectors**

**Connectors** — это механизм подключения ИИ к внешним системам. С их помощью Claude может взаимодействовать с различными сервисами, включая Wildberries, через платформу MarketAut.

![](img/claude/customize-connectors.png)

В открывшемся окне отображается список уже подключённых коннекторов.

4) Для подключения нового коннектора нажмите **(+)**, затем в открывшемся меню выберите **Add custom connector**
![](img/claude/connectors-add.png)

Откроется окно настроек **сonnectors**. 

Заполните следующие значения:
**Имя** — любое название на ваш выбор, например **`MarketAut`**
**Адрес** — **`https://marketaut.ru/mcp/wb`**

После заполнения полей нажмите **Add**.

![](img/claude/connector-dialog.png)


После добавления коннектора **MarketAut** появится в списке, но будет отображаться как неподключённый. 
Нажмите **Connect**, чтобы установить подключение.

![](img/claude/not-connected.png)

Вы будете перенаправлены на страницу авторизации **MarketAut**. 
Нажмите **Разрешить доступ**.

![](img/claude/oauth_consent.png)


После завершения проверки статус коннектора изменится на «**Подключён**».

![](img/claude/connected.png)

# Работа с Claude

## Проверка подключения

В любом диалоге с Claude нажмите на символ **(+)** под полем ввода сообщения и выберите **Connectors**.

Если подключение настроено корректно, в списке **Connectors** будет отображаться **MarketAut** (или другое имя, указанное при настройке подключения).

![](img/claude/check_connectors.png)

## Обновление инструментов

После изменения списка доступных инструментов в веб-интерфейсе MarketAut необходимо обновить список инструментов в Claude.

Для этого перейдите в раздел **Customize**.

![](img/claude/customize.png)

Далее откройте раздел **Connectors**.
![](img/claude/connectors.png)

Выберите нужный коннектор, нажмите на кнопку меню **(⋯)** в правом верхнем углу и выберите **Refresh tools list**.

![](img/claude/refresh-tools.png)

## Настройка доступа к инструментам

Для каждого инструмента можно настроить отдельную политику доступа. 

Доступны следующие варианты:
1. Полный запрет
2. Подтверждение перед каждым использованием
3. Использование без подтверждения

Для настройки перейдите в раздел **Customize** и выберите коннектор **MarketAut**.

![](img/claude/connectors.png)

В правой части окна отображается список инструментов. Для каждого инструмента можно выбрать один из режимов доступа:

![](img/claude/access-mode.png)

## Добавление ресурсов через Add from MarketAut

Если к модулю MCP подключены инструменты, для которых существует skill, в меню Claude появляется пункт **Add from MarketAut**.

Через него можно добавить ресурсы MCP-сервера в текущий диалог или проект. После добавления Claude сможет использовать эти ресурсы при работе с запросами.

![](img/claude/skill-ma.png)

Чтобы добавить ресурс:

1. Нажмите кнопку **+** в поле ввода Claude
2. Выберите **Connectors**
3. Откройте **Add from MarketAut**
4. Выберите нужный ресурс, например **Wildberries**

После добавления ресурс станет доступен в текущем диалоге или проекте.
