# Типы аккаунтов

Тип аккаунта определяет, к какой системе он позволяет подключиться.

**Marketaut** поддерживает следующие типы:

1. Wildberries: API токен [Wildberries](https://wildberries.ru) 


## Wildberries

Позволяет подключаться к магазину [Wildberries](https://wildberries.ru)

**Доступно**
- Карточки товаров
- Цены, скидки, акции
- Фулфилмент
- Вопросы и отзывы покупателей
- Аналитические и финансовые отчеты

#### Получение токена

Перед созданием Аккаунта необходимо получить Токен API Wildberries.
Для этого вы должны быть **владельцем** магазина ВБ.

1) Зайдите в настройки магазина, "[Интеграции по API](https://seller.wildberries.ru/api-integrations)"
2) Нажмите **Создать токен**
3) Выберите **Для интеграции вручную**
4) Выберите **Базовый токен**
5) Выберите разделы, к которым нужен доступ (Контент, Цены и скидки)
6) Нажмите **Создать токен**
7) Скопируйте токен и добавьте его в **Marketaut**
