## Подключение Claude к магазину Wildberries

Для подключения к магазину Wildberries потребуется токен API. [Здесь]() рассказывается, как его выписать. 

1) Создаем Аккаунт для работы с ВБ
	1) Заходим в раздел [Конфигурация->Аккаунты](https://web.marketaut.ru/app/config/accounts
	2) Нажимаем кнопку Добавить Аккаунт
	3) Выбираем Тип Аккаунта - **Wildberries**. После этого у нас появляются новые поля, соответствующие ВБ
		1) Вводим Название и Описание (любые)
		2) Вставляем Токен API (полученный в личном кабинете ВБ)
		3) Жмем Сохранить
